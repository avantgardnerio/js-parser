js-parser
=========

Parses your JavaScript, then dumps it into a graphml, where it can be:

* Imported into a graph database
* Queried by Gremlin
* Rendered with Gephi

![Gephi](Gephi.png)

Usage
-----
java -jar js-parser.jar /path/to/js/root /path/to/output.graphml
